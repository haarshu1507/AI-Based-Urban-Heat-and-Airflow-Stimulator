# AI-based Urban Heat Airflow Simulator

A simulation project focused on urban heat behavior and airflow analysis.

## Tech Stack

- React
- Vite
- Three.js
- Tailwind CSS

## Getting Started

### 1) Clone the repository

```bash
git clone https://github.com/haarshu1507/AI-Based-Urban-Heat-and-Airflow-Stimulator
cd AI-Based-Urban-Heat-and-Airflow-Stimulator
```

### 2) Install dependencies

```bash
cd 2D-Urban
npm install
```

### 3) Run the development server

```bash
npm run dev
```

### 4) Build for production

```bash
npm run build
```
